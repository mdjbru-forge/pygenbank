Welcome to pygenbank's documentation!
=====================================

Description
-----------

This Python module provides some light wrapper functions around Biopython tools
to provide a simplified interface to NCBI's GenBank server. Those functions are
accessible from the ``genbank`` module, and in addition a few scripts
accessible from the command lines are provided.
